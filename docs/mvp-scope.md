# MVP 功能范围

## 英语

| 编号 | 题型 | 首期策略 |
|---|---|---|
| E1 | 单选、多选、判断 | 标准答案自动判分 |
| E2 | 单词、短语填空 | 规范化、多答案与词形规则 |
| E3 | 限定句子填空、句型转换 | 规则判分，边界答案复核 |
| E4 | 阅读简答 | 评分点与语义辅助，首期不自动发布 |

## 数学

| 编号 | 题型 | 首期策略 |
|---|---|---|
| M1 | 数值、分数、小数、百分数 | 精确值、误差与格式检查 |
| M2 | 代数表达式 | 安全 AST、符号等价与形式检查 |
| M3 | 方程与方程解 | 解集与定义域比较 |
| M4 | 分步骤计算 | 相邻步骤验证与首错定位 |

## 非目标

拍照全自动批改、作文自动最终评分、几何证明自动评分、在线直播、课程商城、家长端、原生移动端和在线监考不进入第一阶段。
